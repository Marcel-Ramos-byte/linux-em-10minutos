# Instalar programas

No Ubuntu, usamos `apt`.

Atualizar lista:
sudo apt update - atualiza o banco de dados do sistema
sudo apt upgrade - atualiza o sistema (sempre faça update antes de upgrade)

sudo apt install "programa" - instala "programa"

Exemplo:
sudo apt install git - instala o git na sua maquina

para remover programa
sudo apt remove "programa"

exemplo:
sudo apt remove git - remove git da maquina
