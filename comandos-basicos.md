# Comandos básicos

| comando | o que faz |
|------|------|
| ls | lista arquivos |
| cd | entra em uma pasta |
| pwd | mostra pasta atual |
| mkdir | cria pasta |
| rm | remove arquivo |
| cp | copia |
| mv | move ou renomeia |

---

## Exemplos

Criar pasta:
mkdir projetos - cria a pasta projetos
cd projetos - entra na pasta projetos
