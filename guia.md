# Terminal

O terminal é uma forma de controlar o computador usando texto.

Abra o terminal com CTRL + T

Teste:

pwd - esse comando mostra a pasta atual.

ls  - mostra os arquivos existentes na pasta atual

cd "pasta" - esse comando entra em "pasta"
ou seja voce listou com "ls" e tinha uma pasta chamada joseph e voce quer ver oque tem nela, entao voce vai entrar nela com o comando cd joseph
e depois se quiser ver oque tem na pasta joseph voce digita "ls" novamente e assim sucessivamente

cd .. - esse comando leva voce um nivel de volta doque voce esta
no exemplo da pasta joseph que dei depois de entrar nela voce quer voltar para a pasta que voce estava entao voce digita cd .. e o linux vai voltar para a pasta que voce esta anteriormente
